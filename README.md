# CrochetHub 🧶

CrochetHub is a platform that connects crocheters and customers while simplifying custom order management.

## Features

- Customer Registration
- Crocheter Registration
- Custom Order Requests
- Project Progress Tracking
- Customer Dashboard
- Crocheter Dashboard
- Order Management

## Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
